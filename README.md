# ♟️ Real-Time Chess Game

A real-time multiplayer chess game built using Node.js, Express.js, Socket.IO, and Chess.js.

## Technologies Used

* **Node.js** – JavaScript runtime used to run the server.
* **Express.js** – Web framework for creating the server and handling routes.
* **Socket.IO** – Enables real-time communication between players.
* **Chess.js** – Handles chess rules and validates legal moves.
* **EJS** – Template engine for rendering web pages.
* **HTML** – Creates the structure of the web page.
* **CSS** – Styles the chessboard and user interface.
* **JavaScript** – Adds interactivity and game logic on the client side.
* **Tailwind CSS** – Utility-first CSS framework for responsive styling.

## Features

* Real-time multiplayer gameplay
* Two-player support (White & Black)
* Spectator mode
* Drag-and-drop chess pieces
* Server-side move validation
* Live board synchronization
* Turn-based gameplay
* Automatic board rotation for Black player
